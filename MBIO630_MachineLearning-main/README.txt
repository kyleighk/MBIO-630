----- Hi! Welcome to the Machine Learning module! -------

Here are some instructions to start working with your Machine Learning models.

--- R packages instructions ---

1. Install the following packages in R studio, using the function:

install.packages(c('kohonen', 'ggplot2', 'raster', 'reshape2', 'neuralnet', 'sp', 'vcd')

2. Download the 'kohonen2.tar' file from Github repository (MachineLearning/data/kohonen2.tar),
   then install it in R studio going on Tools->Install packages->Install from Package Archive->your path
